# 🍰 Pasya Cake Website

Official website of **Pasya Cake** – Homemade bakery with online ordering system and email notification integration.

---

## 🌸 About Pasya Cake

Pasya Cake is a homemade bakery offering premium cakes and custom orders.  
Customers can explore the menu, read testimonials, and place orders directly through the website.

---

## 🚀 Features

- 📋 Cake menu list layout
- ⭐ Customer testimonials
- 🛒 Online order form
- 📩 Automatic email notification (EmailJS → Gmail)
- 📍 Google Maps location embed
- 📱 Fully responsive design (Mobile Friendly)
- 🎨 Soft pastel aesthetic design
